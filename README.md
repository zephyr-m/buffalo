# 📋 Мультибуфер обмена

Мультибуфер с историей копирований и быстрым доступом.

## 🚀 Запуск

```bash
cd /home/zephyr/aes/develop/clipboard-manager
./start.sh
```

## 🔥 Использование

- **Ctrl+F** - показать/скрыть окно с историей (работает на любой раскладке)
- **Esc** - скрыть окно
- **Клик на 📋** - скопировать команду в буфер
- **Клик на 🗑️** - удалить запись

## 🛑 Остановка

```bash
pkill -f clipboard_manager.py
```

## ✨ Особенности

- ✅ **Toggle окно** - Ctrl+F показывает/скрывает окно
- ✅ **Независимость от раскладки** - работает на EN/RU
- ✅ **Карточки команд** - каждая запись в отдельном контейнере
- ✅ **Кнопки копирования** - клик на 📋 сразу копирует
- ✅ **Современный дизайн** - тёмная тема, hover эффекты
- ✅ **Автоматическое отслеживание** всех копирований
- ✅ **История до 50 записей** с временем
- ✅ **Сохранение между сессиями**
